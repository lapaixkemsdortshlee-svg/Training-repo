# Training-repo

Preloaded with design/UI skills + the 21st.dev MCP server for professional
website creation. See [CLAUDE.md](CLAUDE.md) for what's installed and how to
configure the `TWENTYFIRST_API_KEY` for the 21st MCP.

## Quick start

```bash
cp .env.example .env   # then set TWENTYFIRST_API_KEY
```

Skills live in `.claude/skills/`; MCP config in `.mcp.json`.
